Documentation

This movie API will give users access to information about their favourite movies, genres, actors, directors etc. Also, users will be able to create a profile, update their personal information and create a list of their favourite movies.

Information on how to utilize the API

Business Logic URL Endpoint HTTP Method Query Parameters Request Data Body Format Response Data Body Format
Return a list of ALL movies to the user /movies GET None None JSON object with data about all movies:
Array of: {"title": "Title of the movie", "year": "year of release", "description": "plot of movie", "genre": "Genre of the movie", "director": "Name of the director", "actors": "name of actors"}
Return movie by title /movies/:title GET None None JSON object with data about a specific movie:
Array of: {"title": "Title of the movie", "year": "year of release", "description": "plot of movie", "genre": "Genre of the movie", "director": "Name of the director", "actors": "name of actors"}
Return movie by genre /movies/genre/:genreName GET None None JSON object with data about movies of a specific genre:
Array of: {"title": "Title of the movie", "year": "year of release", "description": "plot of movie", "genre": "Genre of the movie", "director": "Name of the director", "actors": "name of actors"}
Return movie by director /movies/director/:directorName GET None None JSON object with data about movies of a specific director:
Array of: {"title": "Title of the movie", "year": "year of release", "description": "plot of movie", "genre": "Genre of the movie", "director": "Name of the director", "actors": "name of actors"}
Allow new users to register /users/register POST None JSON object with data to register.
{"Username", "Password", "Email", "Birthday"} 		JSON object sent with registered data plus an additional random unique ID value recorded.
Allow users to update their user info (username) /users/:userName PUT None JSON object of data to update.
{"Username", "Password", "Email", "Birthday"} 		JSON object sent with updated data.
Allow existing users to deregister /users/:userName DELETE None None A text message confirming the delete
Allow users to add a movie to their list of favourites /users/:Username/movies/:MovieID POST None None JSON object with info about favourite movies
{"Username", "Password", "Email", "Birthday", "FavouriteMovies"}
Allow users to remove a movie from their list of favourites /users/:Username/movies/:MovieID DELETE None None JSON object with info about favourite movies
{"Username", "Password", "Email", "Birthday", "FavouriteMovies"}
